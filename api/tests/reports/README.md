# Test Reports

**Doc-Type:** Documentation · Version 1.0.0 · Updated 2025-11-16 · AI Whisperers

## Purpose

This directory contains test reports for the YouTube Transcript Extractor project, including A/B testing results, performance benchmarks, and stability analysis.

---

## Report Types

### Stealth A/B Test Reports

**Purpose:** Compare extraction reliability with stealth techniques enabled vs disabled (raw Chromium)

**Files:**
- `stealth-test-interim-*.md` - Interim progress reports during test execution
- `stealth-test-interim-data-*.json` - Machine-readable interim data
- `stealth-test-results.md` - Final comprehensive analysis (generated by test script)
- `stealth-test-data.json` - Final raw test data (generated by test script)

**Location of test script:** `api/scripts/stealth-ab-test.ts`

**How to run:**
```bash
# Full test (100 iterations per mode, ~2 hours)
cd api
npm run stealth-test

# Quick test (10 iterations per mode, ~10 minutes)
npm run stealth-test:quick
```

## Report Structure

### Markdown Reports (.md)

Human-readable reports following dual-layer documentation pattern:
- Executive summary
- Results comparison tables
- Error breakdown analysis
- Performance metrics
- Recommendations with rationale
- Reproducibility instructions

### JSON Data Files (.json)

Machine-readable test data for further analysis:
- Test configuration and environment
- Per-video success rates
- Detailed failure information
- Statistical summaries
- Predictions and analysis

## Accessing Reports

**From command line:**
```bash
# View interim markdown report
cat api/tests/reports/stealth-test-interim-2025-11-16.md

# Parse interim JSON data
node -e "console.log(JSON.stringify(require('./api/tests/reports/stealth-test-interim-data-2025-11-16.json'), null, 2))"
```

**From code:**
```typescript
import * as fs from 'fs';
import * as path from 'path';

// Load JSON data
const dataPath = path.join(__dirname, 'reports', 'stealth-test-interim-data-2025-11-16.json');
const testData = JSON.parse(fs.readFileSync(dataPath, 'utf-8'));

console.log(`Success rate: ${testData.stealthModeResults.successRate}%`);
console.log(`Tests completed: ${testData.testProgress.completed}/${testData.testProgress.totalPlanned}`);
```
